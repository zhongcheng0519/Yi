# 易经六十四卦网站

一个基于React + TypeScript开发的交互式易经六十四卦查询网站。

## 功能特点

### 🎯 核心功能
- **卦象选择**: 可以分别选择上卦和下卦
- **实时组合**: 自动显示组合后的完整卦象
- **卦辞显示**: 展示对应卦的卦辞和卦意解释
- **爻线表示**: 用实线（阳爻）和断线（阴爻）准确表示卦象

### 🎨 用户体验
- **现代化UI**: 采用渐变背景、玻璃拟态效果
- **响应式设计**: 支持桌面和移动设备
- **直观交互**: 点击选择，实时更新结果
- **视觉反馈**: 选中状态有明显的视觉区分

### 📱 界面特色
- **美观设计**: 渐变背景、毛玻璃效果
- **中文支持**: 完整的中文界面和易经术语
- **动画效果**: 平滑的过渡和淡入动画
- **信息丰富**: 卦名、卦象、卦辞、卦意一应俱全

## 技术架构

### 前端框架
- **React 18**: 现代化的用户界面库
- **TypeScript**: 提供类型安全
- **Vite**: 快速的构建工具

### 项目结构
```
src/
├── App.tsx              # 主应用组件
├── App.css              # 样式文件
├── TrigramSelector.tsx  # 卦象选择器组件
├── HexagramDisplay.tsx  # 卦象显示组件
└── hexagram-data.ts     # 易经数据文件
```

### 核心数据
- **8个基本卦象**: 乾、坤、震、巽、坎、离、艮、兑
- **64个完整卦象**: 包含卦名、卦辞、卦意解释
- **二进制编码**: 每卦用6位二进制表示（1=阳爻，0=阴爻）

## 使用方法

### 安装和运行
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 操作步骤
1. **选择上卦**: 在"上卦选择"区域点击选择卦象
2. **选择下卦**: 在"下卦选择"区域点击选择卦象
3. **查看结果**: 自动显示组合后的卦象信息
4. **重置选择**: 点击"重置选择"按钮清除选择

### 卦象选择
- 每个选择器显示8个基本卦象
- 卦象包含：符号、名称、含义描述
- 选中的卦象会有明显的视觉反馈

## 项目特色

这个易经网站不仅是一个功能性的工具，更是一个融合了传统文化与现代技术的作品。通过现代化的Web技术，让古老的易经智慧以新的形式呈现给用户，既保持了传统文化的庄重感，又具备了现代应用的用户体验。

网站的设计充分考虑了易经文化的神秘感和厚重感，同时又保持了简洁易用的界面。用户可以通过这个网站轻松学习和了解易经六十四卦的组合规律和含义。
